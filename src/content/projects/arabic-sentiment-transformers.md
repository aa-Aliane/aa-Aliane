## Project Overview

Exploited Transformer-based architectures (BERT, AraBERT) via the Hugging Face library to build high-accuracy sentiment analysis models for the Arabic language.

## Technical Details

- **Architectures**: BERT, AraBERT, mBERT.
- **Frameworks**: PyTorch, Hugging Face Transformers.
- **Languages**: Arabic (MSA and Dialectal).
- **Outcome**: Improved classification accuracy compared to traditional machine learning baselines.
